📦 Part 1: What is Pandas?

Pandas is a powerful and popular Python library for data manipulation and analysis. It provides fast, flexible, and expressive data structures designed to make working with "relational" or "labeled" data both easy and intuitive.

Analogy: If NumPy is the superhighway for numbers, Pandas is the ultimate smart spreadsheet (like Excel or Google Sheets) for Python. It's built on top of NumPy, but it gives us the ability to work with complex, tabular data (with rows and columns), use descriptive labels, and handle missing data gracefully.

Importing Pandas

By universal convention, Pandas is always imported with the alias pd.

import pandas as pd
import numpy as np # Pandas often works alongside NumPy

📊 Part 2: Core Pandas Data Structures

Pandas has two fundamental data structures: the Series and the DataFrame.

The Series

A Series is a one-dimensional, labeled array. It's like a single column in a spreadsheet.

  • Data: The actual values (a NumPy array).
  • Index: A sequence of labels for the data. If not specified, it defaults to integers 0, 1, 2, ....
# Create a file named day4_series.py
import pandas as pd

# A simple Series with a default index
s = pd.Series([1, 2, 5, 6])
print("Simple Series:")
print(s)

# A Series with a custom index
populations = pd.Series([38, 26, 19], index=['California', 'Texas', 'New York'])
print("\nSeries with custom index:")
print(populations)

# Accessing data using the index
print(f"\nPopulation of Texas: {populations['Texas']}")

The DataFrame

A DataFrame is a two-dimensional, labeled data structure with columns of potentially different types. It is the most commonly used Pandas object. It's like a full spreadsheet or an SQL table.

  • You can think of a DataFrame as a dictionary of Series, where the keys are the column names and the values are the Series (the columns).
# Create a file named day4_dataframe.py
import pandas as pd

# Creating a DataFrame from a dictionary of lists
data = {
    'State': ['California', 'Texas', 'New York', 'Florida'],
    'Population': [38, 26, 19, 19],
    'Capital': ['Sacramento', 'Austin', 'Albany', 'Tallahassee']
}

df = pd.DataFrame(data)
print(df)

🎯 Part 4: Selecting and Filtering Data

Selecting subsets of your data is a core task in data analysis. Pandas provides two main methods for this: .loc[] and .iloc[].

  • .loc[]: Selects data by label (the index name and column name).
  • .iloc[]: Selects data by integer location (the row and column number, starting from 0).

Selecting Columns

# Create a file named day4_selecting.py
import pandas as pd

data = {
    'State': ['California', 'Texas', 'New York', 'Florida'],
    'Population': [38, 26, 19, 19],
    'Capital': ['Sacramento', 'Austin', 'Albany', 'Tallahassee']
}
df = pd.DataFrame(data)

# Select a single column (returns a Series)
population_series = df['Population']
print("Population Series:")
print(population_series)

# Select multiple columns (returns a DataFrame)
state_capital_df = df[['State', 'Capital']]
print("\nState and Capital DataFrame:")
print(state_capital_df)

Selecting Rows with .loc and .iloc

# In day4_selecting.py

# Set the 'State' column as the index for better label-based access
df = df.set_index('State')

# Select a single row by its index label using .loc
texas_data = df.loc['Texas']
print("\nData for Texas (.loc):")
print(texas_data)

# Select a single row by its integer position using .iloc
first_row = df.iloc[0]
print("\nData for the first row (.iloc):")
print(first_row)

Filtering with Boolean Indexing

This is the most common way to select rows that meet a certain condition. It works just like with NumPy arrays.

# In day4_selecting.py

# Find all states with a population greater than 20 million
high_population_states = df[df['Population'] > 20]
print("\nStates with population > 20M:")
print(high_population_states)

Assignment Solution (sales_analyzer.py)

# Day_4_sol.py
# Solution for the Day 4 "Basic Sales Data Analysis" assignment.

import pandas as pd

# 1. Read the CSV file.
try:
    sales_df = pd.read_csv("sales_data.csv")

    # 2. Print the entire DataFrame.
    print("--- Full Sales Data ---")
    print(sales_df)

    # 3. Select and print 'Product' and 'Quantity' columns.
    print("\n--- Product and Quantity ---")
    product_quantity = sales_df[['Product', 'Quantity']]
    print(product_quantity)

    # 4. Filter for the 'North' region.
    print("\n--- Sales in the North Region ---")
    north_sales = sales_df[sales_df['Region'] == 'North']
    print(north_sales)

    # 5. Filter for quantity > 100.
    print("\n--- Sales with Quantity > 100 ---")
    large_sales = sales_df[sales_df['Quantity'] > 100]
    print(large_sales)

    # 6. Filter for Apples with Price > 1.2.
    print("\n--- Expensive Apple Sales ---")
    expensive_apples = sales_df[(sales_df['Product'] == 'Apples') & (sales_df['Price'] > 1.2)]
    print(expensive_apples)

except FileNotFoundError:
    print("Error: 'sales_data.csv' not found. Please create it first.")
