📦 Part 1: What is a List?

A list is an ordered and changeable collection of items. They allow you to store multiple items in a single variable.

Analogy: Think of a list as a train. Each car is an item, and they stay in the order you put them in. You can add new cars, remove old ones, or change what's inside a car at any time.

Creating a List

Lists are created using square brackets [], with items separated by commas.

# Create a file named day6_basics.py

# A list of numbers
high_scores = [100, 95, 95, 88, 70]
print(high_scores)

# A list of strings
class_subjects = ["Data Science", "Python", "AI"]
print(class_subjects)

# A list with mixed data types
student_info = ["Alex", 25, True, 1.75]
print(student_info)

🔎 Part 2: Accessing List Items

Indexing

You can access any item in a list by its index (position), which starts at 0.

Visual Guide to Indexing subjects:

 ["Data Science", "Python", "AI"]
       0             1        2      (Positive Index)
      -3            -2       -1      (Negative Index)
```python
# Create a file named day6_accessing.py
subjects = ["Data Science", "Python", "AI"]

# Positive Indexing (from the start)
print(f"The first subject is: {subjects[0]}")

# Negative Indexing (from the end)
print(f"The last subject is: {subjects[-1]}")

Slicing

Slicing lets you get a sub-list (a portion of the original list).

Syntax: list[start:end:step]

# In day6_accessing.py
tasks = ['Task 1', 'Task 2', 'Task 3', 'Task 4', 'Task 5']

# Get items from index 1 up to (but not including) index 4
print(f"Middle tasks: {tasks[1:4]}") # Output: ['Task 2', 'Task 3', 'Task 4']

# Get items from the start up to index 3
print(f"First three tasks: {tasks[:3]}")

# Get every second item from the list
print(f"Alternate tasks: {tasks[::2]}") # Output: ['Task 1', 'Task 3', 'Task 5']

🔧 Part 3: Modifying Lists

Because lists are mutable, we can change them after they are created.

Adding Items to a List

MethodDescription
append(item)Adds a single item to the end of the list.
insert(index, item)Inserts an item at a specific position.
extend(iterable)Adds all items from another list (or tuple/set) to the end.
# Create a file named day6_modifying.py
playlist = ["Song A", "Song B", "Song C"]
print(f"Original Playlist: {playlist}")

# .append()
playlist.append("Song D")
print(f"After append: {playlist}")

# .insert()
playlist.insert(1, "New Favorite Song")
print(f"After insert: {playlist}")

# .extend()
more_songs = ["Song E", "Song F"]
playlist.extend(more_songs)
print(f"After extend: {playlist}")

Removing Items from a List

Method / KeywordDescription
remove(item)Removes the first matching item by its value.
pop(index)Removes an item by its index and returns it. If no index, removes the last item.
del list[index]A keyword that removes an item by its index.
clear()Removes all items from the list, making it empty.

Changing an Item

You can change an item by accessing its index and assigning a new value.

# In day6_modifying.py
shopping_list = ["apples", "oranges", "bread"]
print(f"\nOriginal shopping list: {shopping_list}")

# Change an item
shopping_list[1] = "bananas"
print(f"After changing: {shopping_list}")

✨ Part 4: More Powerful List Methods

Sorting and Reversing

MethodDescription
sort()Sorts the original list in place (ascending by default).
sort(reverse=True)Sorts the original list in descending order.
reverse()Reverses the order of elements in the original list.

Important: sort() modifies the list directly. It does not return a new list.

Other Useful Methods

MethodDescription
len(list)(A function, not a method) Returns the number of items in the list.
count(item)Returns how many times an item appears in the list.
index(item)Returns the index of the first occurrence of an item.
copy()Returns a shallow copy of the list.

⚡ Part 5: List Comprehension (A Pro Shortcut)

List comprehension is an elegant way to create lists based on existing lists. It's a compact version of a for-loop.

Syntax: newList = [expression for item in oldList if condition]

# Create a file named day6_comprehension.py

# Traditional way to get numbers squared
numbers = [1, 2, 3, 4, 5]
squared_numbers = []
for num in numbers:
    squared_numbers.append(num * num)
print(f"Traditional way: {squared_numbers}")

# Using List Comprehension
comp_squared = [num * num for num in numbers]
print(f"Comprehension way: {comp_squared}")

# With a condition: only get even numbers
even_numbers = [num for num in numbers if num % 2 == 0]
print(f"Even numbers only: {even_numbers}")
