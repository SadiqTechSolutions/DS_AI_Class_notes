üí¨ Part 1: String Basics

What is a String?

A string is a sequence of characters enclosed in quotes. You can use either single (') or double (") quotes.

Analogy: Think of a string as a necklace, where each character (letter, number, symbol) is a bead.

# Create a file named day5_basics.py
greeting = "Hello, World!"
user_name = 'Python_Student_123'

print(greeting)
print(user_name)

Multi-line Strings

For text that spans multiple lines, use triple quotes (""" or ''').

# In day5_basics.py
instructions = """
Welcome to Day 5.
Today, we will be mastering strings.
Let's begin!
"""
print(instructions)

Quotes Inside Strings

To put a quote inside a string, you can use the opposite type of quote for the container, or use an escape character.

# In day5_basics.py
# Using opposite quotes
sentence1 = 'He said, "Python is amazing!"'
print(sentence1)

# Using an escape character (\)
sentence2 = "He said, \"Python is amazing!\""
print(sentence2)

Escape Characters

These are special characters that let you insert characters you can't type directly.

CharacterDescriptionExample
\'Single Quote'It\'s me.'
\"Double Quote"She said \"Hi\"."
\\Backslash"C:\\Users\\..."
\nNew Line"Line 1\nLine 2"
\tTab"Hello\tWorld"

‚úÇÔ∏è Part 2: String Operations - Slicing and Dicing

String Length: len()

The len() function tells you how many characters are in a string.

# Create a file named day5_operations.py
topic = "Data Science"
print(f"The length of '{topic}' is {len(topic)} characters.") # Output: 12

Strings as Arrays: Indexing & Slicing

Strings are ordered sequences, so you can access individual characters by their index (position), starting from 0.

Slicing lets you extract a "slice" of the string.

Visual Guide to Slicing python:

 p  y  t  h  o  n
 0  1  2  3  4  5  (Positive Index)
-6 -5 -4 -3 -2 -1  (Negative Index)
```python
# In day5_operations.py
word = "python"

# Indexing
print(f"First character: {word[0]}")  # 'p'
print(f"Last character: {word[-1]}") # 'n'

# Slicing
print(f"Slice from index 2 to 5: {word[2:5]}") # 'tho' (index 5 is not included)
print(f"Slice from start to index 3: {word[:3]}") # 'pyt'
print(f"Slice from index 3 to end: {word[3:]}")   # 'hon'

‚úçÔ∏è Part 4: Formatting Strings

Method 1: Concatenation (+)

The simplest way, but can be clumsy with non-string types.

# Create a file named day5_formatting.py
first_name = "Alex"
last_name = "Smith"
age = 30

# Clumsy concatenation
full_description = "User: " + first_name + " " + last_name + ". Age: " + str(age)
print(full_description)

Method 2: .format() Method

More flexible, uses placeholders {}.

# In day5_formatting.py
# .format() method
statement = "User: {}. Age: {}."
print(statement.format(first_name, age))

Method 3: f-Strings (The Modern Way)

F-strings (formatted string literals) are the easiest and most powerful way. Start the string with an f and put variables directly inside curly braces {}.

# In day5_formatting.py
# f-string (recommended)
f_string_statement = f"User: {first_name} {last_name}. Age: {age}."
print(f_string_statement)

Assignment Solution (log_formatter.py)

# Day_5_sol.py
# Solution for the Day 5 "Log File Formatter" assignment.

# The raw log entry string
log_entry = "INFO:2025-07-04T09:54:01Z;User 'alex_p' logged in successfully from IP 192.168.1.101.   "

# 1. Clean up trailing whitespace.
cleaned_log = log_entry.strip()

# 2. Extract the log level.
colon_index = cleaned_log.find(':')
log_level = cleaned_log[:colon_index]

# 3. Extract the timestamp.
semicolon_index = cleaned_log.find(';')
timestamp = cleaned_log[colon_index + 1 : semicolon_index]

# 4. Extract the username.
start_quote_index = cleaned_log.find("'")
end_quote_index = cleaned_log.find("'", start_quote_index + 1)
username = cleaned_log[start_quote_index + 1 : end_quote_index]

# 5. Extract the IP address.
ip_start_index = cleaned_log.find("IP ") + 3 # Add 3 to get past "IP "
ip_address = cleaned_log[ip_start_index:-1] # Slice until the last character (the period)

# 6. Extract the message.
message_start_index = end_quote_index + 2 # After the closing quote and space
message_end_index = cleaned_log.find(" from IP")
message = cleaned_log[message_start_index:message_end_index]

# 7. Format and print the report with transformations.
print("--- Log Analysis Report ---")
print(f"Timestamp: {timestamp[:10]}")      # Slice the timestamp to get just the date
print(f"Log Level: {log_level}")
print(f"Username:  {username.upper()}")    # Convert username to uppercase
print(f"IP Address:{ip_address}")
print(f"Message:   {message.capitalize()}") # Capitalize the message
print("---------------------------")
