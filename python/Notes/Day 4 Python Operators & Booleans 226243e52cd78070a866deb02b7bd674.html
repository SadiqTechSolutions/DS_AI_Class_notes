🧮 Part 1: Python Operators

What are Operators?

Operators are special symbols in Python that carry out arithmetic or logical computation.

Analogy: Think of operators as the verbs of programming. They are the action words that let you do things with your variables (the nouns).

Used to perform mathematical operations.

✅ Part 2: Booleans and "Truthiness"

What are Booleans?

A Boolean is a data type that can only have one of two values: True or False. They are the foundation of decision-making in code.

# Create a file named day4_booleans.py
can_vote = False
age = 20

if age >= 18:
    can_vote = True

print(f"Is the user eligible to vote? {can_vote}")

The Concept of "Truthiness"

In Python, every value can be evaluated as either "truthy" or "falsy" in a boolean context.

  • Falsy: Values that evaluate to False.
  • Truthy: Values that evaluate to True.

The only "Falsy" values in Python are:

  • The number 0 (and 0.0)
  • An empty string ""
  • An empty list [], tuple (), dictionary {}, or set
  • The special value None

Everything else is "Truthy"!

We can check a value's truthiness with the bool() function.

# In day4_booleans.py
print(f"Is 0 True or False? {bool(0)}")          # False
print(f"Is 'hello' True or False? {bool('hello')}") # True
print(f"Is [] True or False? {bool([])}")        # False
print(f"Is [1, 2] True or False? {bool([1, 2])}")  # True

💡 Interesting Facts

  • Logical operators and and or use "short-circuiting". If the result can be determined from the first operand (e.g., False and ...), Python won't even evaluate the second one!
  • The is operator checks if two variables point to the exact same object in memory, while == checks if their values are equal. For beginners, == is almost always what you want.
  • Operator precedence is why 2 + 3 * 4 equals 14, not 20. Python follows the standard order of operations (PEMDAS/BODMAS).
  • In Python, True is internally represented as the integer 1, and False as 0. You can even do math with them: True + True equals 2!

📝 Day 4 Assignment Solution

Assignment Solution:

# Day_4_sol.py
# Solution for the Day 4 "Simple Store Checkout Calculator" assignment.

# 1. Define variables for item prices.
item1_price = 10.50
item2_price = 20.00
item3_price = 5.25

# 2. Calculate the subtotal.
subtotal = item1_price + item2_price + item3_price

# 3. Define membership status.
has_membership = True

# 4. Determine discount eligibility using logical 'and'.
is_eligible_for_discount = subtotal >= 30.00 and has_membership

# 5. Calculate the final total.
# This is a simple way to handle it before learning if-statements.
# We assume we know the eligibility and apply the discount manually.
# In a real program, an if-statement would be used here.
discount_rate = 0.10 # 10% discount

# For the solution, we'll demonstrate the logic as if we're using an if-statement.
if is_eligible_for_discount:
    discount_amount = subtotal * discount_rate
    final_total = subtotal - discount_amount
else:
    final_total = subtotal

# 6. Print the results in a receipt format.
print("--- STS Store Receipt ---")
print(f"Item 1 Price: ${item1_price:.2f}")
print(f"Item 2 Price: ${item2_price:.2f}")
print(f"Item 3 Price: ${item3_price:.2f}")
print("-------------------------")
print(f"Subtotal:     ${subtotal:.2f}")
print("-------------------------")
print(f"Discount Applied: {is_eligible_for_discount}")
if is_eligible_for_discount:
    print(f"Discount (10%): -${discount_amount:.2f}")
print("-------------------------")
print(f"Final Total:  ${final_total:.2f}")
print("-------------------------")
print("Thank you for shopping!")
